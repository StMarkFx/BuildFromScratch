# Phase 1 Core ML Papers & References

This folder contains key research papers, articles, and references for Phase 1 topics. These foundational papers provide the mathematical and theoretical background for the algorithms implemented in the notebooks.

## üìö Key Papers by Topic

### 1. Linear Regression & Optimization
- **"An Introduction to Statistical Learning" (James, Witten, Hastie, Tibshirani)** - Chapter 3: Linear Regression
  - Foundational text covering OLS derivation and statistical interpretation
  - [Book Link](https://www.statlearning.com/)

- **"Pattern Recognition and Machine Learning" (Bishop)** - Chapter 3: Linear Models for Regression
  - Bayesian perspective on linear regression
  - Comprehensive treatment of regularization

### 2. Logistic Regression & Classification
- **"Pattern Recognition and Machine Learning" (Bishop)** - Chapter 4: Linear Models for Classification
  - Theoretical foundation for logistic regression
  - Maximum likelihood estimation derivation

### 3. Decision Trees & Random Forests
- **"Classification and Regression Trees" (Breiman et al., 1984)**
  - Original CART algorithm paper
  - Foundational work on decision tree algorithms

- **"Random Forests" (Breiman, 2001)**
  - Original random forest paper
  - Bootstrap aggregation and random feature selection

- **"Bagging Predictors" (Breiman, 1996)**
  - Theoretical foundation for ensemble methods

### 4. K-Means & Clustering
- **"A Method for Obtaining Digital Signatures and Public-Key Cryptosystems" (Lloyd, 1982)**
  - Original k-means algorithm description

- **"Some Methods for Classification and Analysis of Multivariate Observations" (Hartigan & Wong, 1979)**
  - K-means convergence properties

### 5. Naive Bayes & SVM
- **"A Training Algorithm for Optimal Margin Classifiers" (Bosher et al., 1992)**
  - Early SVM formulation

- **"Support-Vector Networks" (Cortes & Vapnik, 1995)**
  - Modern SVM formulation with kernel trick

### 6. PCA & Dimensionality Reduction
- **"Principal Component Analysis" (Hotelling, 1933)**
  - Original PCA formulation

- **"Principal Component Analysis" (Jolliffe, 2002)**
  - Comprehensive treatment of PCA methods and applications

### 7. Regularization Techniques
- **"Ridge Regression: Biased Estimation for Nonorthogonal Problems" (Hoerl & Kennard, 1970)**
  - Original ridge regression paper

- **"Regression Shrinkage and Selection via the Lasso" (Tibshirani, 1996)**
  - Original LASSO formulation

### 8. Gradient Descent Optimization
- **"Adam: A Method for Stochastic Optimization" (Kingma & Ba, 2014)**
  - Original Adam optimizer paper
  - [arXiv Link](https://arxiv.org/abs/1412.6980)

- **"On the Convergence of Adam and Beyond" (Reddi et al., 2018)**
  - Analysis of Adam convergence properties

## üìñ Recommended Textbooks

1. **"An Introduction to Statistical Learning" (ISLR)** - Free PDF available
   - Practical focus with mathematical rigor
   - Excellent for Phase 1 algorithms

2. **"Pattern Recognition and Machine Learning" (PRML) by Christopher Bishop**
   - Theoretical depth with Bayesian perspective
   - Comprehensive coverage of ML fundamentals

3. **"Machine Learning: A Probabilistic Perspective" (Murphy)**
   - Advanced theoretical treatment
   - Excellent for mathematical foundations
